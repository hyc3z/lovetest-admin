# 导出功能快速指南

## 🚀 快速开始

### 导出全部
1. 点击"导出"按钮
2. 留空（不输入任何内容）
3. 点击"预览匹配结果"
4. 点击"导出激活码"

### 导出特定内容
1. 点击"导出"按钮
2. 输入模式（如：TEST）
3. 点击"预览匹配结果"
4. 确认后点击"导出激活码"

## 📝 常用模式

### 模糊匹配（不勾选正则）
```
TEST        → 包含 TEST 的所有激活码
2024        → 包含 2024 的所有激活码
DEMO-       → 包含 DEMO- 的所有激活码
```

### 正则表达式（勾选正则）
```
^TEST-.*    → 以 TEST- 开头
.*-2024$    → 以 -2024 结尾
^DEMO-\d+$  → DEMO- 后跟数字
```

## 🎯 预览显示

```
找到 156 个匹配的激活码

预览（前10条和最后1条）：
1. CODE-001
2. CODE-002
...
10. CODE-010
... (145 项)
156. CODE-156
```

## 📦 导出文件

**文件名格式：**
```
activation-codes-{模式}-{时间}.txt
```

**示例：**
- `activation-codes-2024-11-21T10-30-00.txt`
- `activation-codes-TEST-2024-11-21T10-30-00.txt`
